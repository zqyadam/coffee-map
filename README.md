# Restaurant Map

这是优达学城前端纳米学位第5个项目——街区地图。

该项目以寻找城市餐厅为主题，通过IP地址定位当前的城市，在地图上展现该城市的餐厅位置，通过左侧下拉列表可以筛选餐厅类型，点击Marker可以显示相关餐厅信息。使用和风天气API获取并展示城市实时天气。

## Demo
https://zqyadam.github.io/restaurant-map/

## 使用
### 安装
下载后
```
yarn install
```

### 启动
```
yarn start
```
